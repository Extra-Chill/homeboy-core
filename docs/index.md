# Homeboy CLI Documentation

Homeboy is a CLI tool for development and deployment automation.

## Quick Start

```bash
homeboy --version
homeboy projects
homeboy docs project
```

## Documentation Topics

- `projects` - List configured projects
- `project` - Project configuration overview
- `project subcommands` - Detailed `project` subcommands
- `server` - Server configuration
- `ssh` - SSH helpers
- `wp` - WP-CLI helpers
- `pm2` - PM2 helpers
- `db` - Database helpers
- `deploy` - Deploy components
- `component` - Component management
- `file` - Remote file operations
- `logs` - Log viewing
- `pin` - Pin frequently used items
- `module` - Module runner
- `git` - Git helpers

For full CLI help for any command, you can also use:

```bash
homeboy help <command>
```
